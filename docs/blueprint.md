# **App Name**: Sarh

## Core Features:

- Digital Asset Registry: Comprehensive tracking for buildings, apartments, and commercial units with real-time status updates, managed via a PostgreSQL database using Prisma.
- Unified Tenant Ledger: Centralized directory for tenant profiles, identity verification storage, and contact history.
- AI Lease Advisor Tool: A generative AI tool that analyzes property details and local market data to generate optimized rental terms and automated contract summaries.
- Payment Lifecycle Tracker: Dashboard for recording rent payments, tracking outstanding arrears, and viewing income statements.
- PDF Contract Engine: Dynamic generation and storage of PDF rental agreements for instant download or archiving.
- Strategic Dashboard: A high-level visual overview of occupancy rates, total portfolio value, and upcoming contract expirations.

## Style Guidelines:

- Primary color: Deep Azure (#1D4ED8) symbolizing professional stability and growth, designed for a modern light-mode management interface.
- Background color: Cool Slate Tint (#F1F5F9), a heavily desaturated variant of the primary hue for a crisp, organized appearance.
- Accent color: Vibrant Indigo (#4F46E5), an analogous hue that provides high contrast for call-to-action buttons and critical alerts.
- A sophisticated font pairing using 'Inter' for bold, machined headlines to denote authority, and 'PT Sans' for body text to maintain warmth and readability in data-heavy lists.
- Minimalist, line-based icons with geometric weight to match the machined look of the Inter typography.
- A bento-style dashboard layout inspired by award-winning portfolio pieces, using generous white space and distinct card modules to separate property types and tenant data.
- Smooth micro-interactions such as layout transitions and soft entry fades to guide the user through complex property listings.
