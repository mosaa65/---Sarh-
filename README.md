## 📖 حول المشروع (About)
**صرح** هو تطبيق ويب حديث متكامل مبني باستخدام إطار العمل **Next.js** ومنظومة **Firebase** السحابية. تم تصميم النظام لتسهيل عملية إدارة العقارات والمستأجرين على الملاك والشركات العقارية، مع توفير ربط محاسبي ذكي وتوليد تلقائي لعقود الإيجار باستخدام تقنيات الذكاء الاصطناعي التوليدي عبر **Genkit**.

---

## 🌟 الميزات الرئيسية (Key Features)

### 📊 لوحة تحكم Bento الذكية (Bento Dashboard)
* **إحصاءات حية**: عرض إجمالي العقارات، الوحدات، والمستأجرين النشطين بنسب إشغال دقيقة.
* **المستحقات المتأخرة**: تتبع ديناميكي لكافة ديون المستأجرين النشطين مستحقة الدفع فوراً.
* **مؤشر التحصيل المالي**: احتساب نسبة التحصيل الشهري الفعلي مقارنة بالدخل المتوقع ديناميكياً.
* **تنبيهات العقود**: رصد وتنبيه تلقائي للعقود القريبة من الانتهاء في الذاكرة لتسهيل التجديد.

### 🏠 سجل الأصول العقارية (Digital Asset Registry)
* تتبع شامل للمباني والشقق والوحدات التجارية مع إمكانية إضافة وإدارة تفاصيل كل أصل عقاري في قاعدة بيانات **Firestore** السحابية الآمنة.

### 👥 السجل المالي للمستأجرين (Unified Tenant Ledger)
* ملف متكامل لكل مستأجر يوضح تفاصيل العقد، الاتصال المباشر عبر WhatsApp، مبالغ التأمين المحجوزة، وتتبع الفواتير الخدمية (كهرباء ومياه) والمديونيات السابقة.

### ✍️ مستشار العقود الذكي بالذكاء الاصطناعي (AI Lease Advisor)
* توليد مسودات قانونية شاملة واحترافية لعقود الإيجار بناءً على بيانات العقار والمستأجر والشروط المطلوبة عبر **Genkit** ونموذج **Gemini 2.5 Flash**.
* **الربط المالي التلقائي**: عند اعتماد العقد، يقوم النظام تلقائياً بتحديث إحصاءات العقار وحساب المستأجر.
* **طباعة نظيفة وتصدير PDF**: واجهة مهيأة بالكامل للطباعة التلقائية أو الحفظ كـ PDF رسمي بدون عناصر الموقع الجانبية أو الرأسية (`print:hidden`).

### 💳 التحصيل المالي وسندات القبض (Payment Lifecycle Tracker)
* إصدار وإدارة سندات القبض بنقرة زر (تحويل بنكي، مدى، نقدي) وتحديث فوري لمديونية المستأجر وتقارير التحصيل في لوحة التحكم.

### 📈 التقارير المالية الذكية (Financial Reports Summary)
* تحليل فوري للنصوص والجداول المالية عبر الذكاء الاصطناعي التوليدي لاستخلاص مؤشرات الأداء، الاتجاهات الإيجابية، والتنبيه للمخاطر والمشاكل المحتملة.

## 🛠️ التقنيات المستخدمة (Tech Stack)

* **إطار العمل الرئيسي**: [Next.js 15](https://nextjs.org/) (App Router & React 19)
* **قاعدة البيانات والهوية**: [Firebase](https://firebase.google.com/) (Authentication & Cloud Firestore)
* **إطار عمل الذكاء الاصطناعي**: [Firebase Genkit](https://firebase.google.com/docs/genkit)
* **نموذج الذكاء الاصطناعي**: [Google Gemini 2.5 Flash](https://ai.google.dev/)
* **التنسيق والجماليات**: [Tailwind CSS](https://tailwindcss.com/) مع استخدام [shadcn/ui](https://ui.shadcn.com/)
* **المكتبات المساعدة**: `date-fns` لإدارة التواريخ، و `lucide-react` للأيقونات الهندسية الأنيقة.

## 🚀 التشغيل المحلي (Getting Started)

تأكد من تثبيت بيئة عمل **Node.js (الإصدار 20 أو أعلى)** على جهازك.

### 1. استنساخ المستودع (Clone the Repository)
```bash
git clone https://github.com/USERNAME/sarh-property-management.git
cd sarh-property-management
```

### 2. إعداد المتغيرات البيئية (Environment Variables)
قم بإنشاء ملف باسم `.env` في المجلد الرئيسي للمشروع وأضف مفتاح **Gemini API**:
```env
GEMINI_API_KEY=AIzaSy...yourKey...
```

### 3. تثبيت المكتبات والاعتمادات (Install Dependencies)
```bash
npm install
```

### 4. بدء تشغيل خادم التطوير (Run Development Server)
```bash
npm run dev
```

افتح متصفحك وتوجه مباشرة إلى الرابط المحلي: **[http://localhost:3000](http://localhost:3000)**

### 5. بناء نسخة الإنتاج (Production Build)
```bash
npm run build
npm run start
```
